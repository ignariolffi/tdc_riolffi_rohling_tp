¿Qué representa cada escenario?
  • Escalón de subida: se enciende una luz y el controlador baja el brillo de la pantalla.
  • Escalón de bajada: se apaga una luz y el controlador aumenta el brillo.
  • Ruido: pequeñas variaciones de iluminación; el controlador intenta mantener el brillo estable.
  • Saturación: hay tanta luz ambiente que el sistema ya no puede compensarla.
  • Flash: aparece una luz intensa durante unos segundos y luego el sistema vuelve a estabilizarse.
